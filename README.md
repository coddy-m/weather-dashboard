### ⛅Weather App
- A `sleek and responsive weather application that provides real-time updates and forecasts for cities worldwide. This project was built to help users plan their day with accurate, up-to-date meteorological data.
### 🚀 Features
- Real-time Weather: Get current temperature, humidity, and wind speed for any city.
Search Functionality: Quickly find weather details by city name.
Dynamic Backgrounds: UI changes based on current weather conditions (e.g., sunny, rainy, cloudy).

- Responsive Design: Fully optimized for desktop, tablet, and mobile devices.
 ### 🛠️ Tech Stack
- Frontend: HTML5, CSS3, JavaScript (ES6+)
- API: OpenWeatherMap API (or specify the one you used)
- Icons: FontAwesome / Weather Icon

### 📖 Usage
- Enter the name of a city in the search bar.
- Press Enter or click the Search icon.
- View the current weather, "feels like" temperature, and high/lows for the day.
